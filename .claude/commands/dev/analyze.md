---
description: Analyze code for quality, security, and performance issues
---

Perform comprehensive code analysis:

## Code Quality
- Check for code smells
- Review naming conventions
- Assess code organization
- Look for duplicate code

## Security
- Check for common vulnerabilities (XSS, SQL injection, etc.)
- Review authentication/authorization
- Check for hardcoded secrets
- Verify input validation

## Performance
- Identify inefficient algorithms
- Look for memory leaks
- Check for unnecessary renders/computations
- Review database query efficiency

## Best Practices
- Check error handling
- Review logging practices
- Assess test coverage
- Verify documentation

Provide specific, actionable recommendations for improvements.
